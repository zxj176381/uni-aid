# Uni-aid

uni-aid 基于 uni-app 开发的一套路由套件，优化开发者的开发体验，在使用前务必请先翻阅文档。

[<div>
  <button class="goDoc">
    快速开始
  </button>
</div>](/uni-aid/pages/guide)

```bash
# 安装
npm install --save-dev uni-aid # 或者：yarn add uni-aid --save-dev

# 命令
npx uniaid init # 初始化目录

npx uniaid add # 创建新的目录或组件

npx uniaid watch # 监听页面文件夹下 index.vue 以及 _uniaid 文件夹下文件的变化。
```
