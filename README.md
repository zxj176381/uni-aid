# uni-aid : 此套件仿 [vni](https://github.com/xiaoyucoding/vni)

uni-aid: 一个用于提升 uni-app 开发体验的扩展套件。

## 更新日志
***
#### 2021/01/14

- 【新增】init 命令：创建页面、页面 json 配置、页面 helper.js、service 以及 routers/alias.js。
